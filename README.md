# ZeroPoint Engine

ZeroPoint Engine is a modular, extensible framework for building custom operating systems from the ground up. Drawing inspiration from the flexibility and power of game engines, ZeroPoint allows developers to construct highly specialized OS environments using a set of reusable, configurable core components.
